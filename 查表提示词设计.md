### 1\. 内阻异常电芯汇总表（dcr\_abnormal\_cells）

__表介绍__

记录每个 BMU 下被判定为内阻异常的电芯。关键列：

- bmu\_code：BMU 编号
- cell\_index：电芯在 BMU 中的位置
- data\_days / abnormal\_days：总数据天数 / 异常天数
- period\_r0\_median\_ohm：周期内阻中位数（欧姆）
- period\_iqr\_ohm：周期内阻四分位距
- first\_abnormal\_time / last\_abnormal\_time：首次/最近异常时间

__示例问题与答案：__

1. 查询某个 BMU 下所有内阻异常的电芯，按异常天数从多到少排序。

SELECT cell\_index, abnormal\_days, period\_r0\_median\_ohm

FROM dcr\_abnormal\_cells

WHERE bmu\_code = '目标BMU编码'

ORDER BY abnormal\_days DESC;

1. 找出最近一周内阻异常天数超过 5 天的电芯。

SELECT bmu\_code, cell\_index, abnormal\_days, last\_abnormal\_time

FROM dcr\_abnormal\_cells

WHERE last\_abnormal\_time >= DATE\_SUB\(NOW\(\), INTERVAL 7 DAY\)

AND abnormal\_days > 5;

1. 统计周期 R0 中位数大于 0\.5 欧姆的电芯数量。

SELECT COUNT\(\*\) AS high\_r0\_cell\_count

FROM dcr\_abnormal\_cells

WHERE period\_r0\_median\_ohm > 0\.5;

### 2\. ISC 评分表（isc\_score\_result）

__表介绍__

存储每个电芯在不同时间窗口的微短路（ISC）综合评分及子分。关键列：

- bmu\_code：BMU 编码
- window\_id：时间窗口标识（如 20230201\_20230801）内短路是按照窗口来诊断的
- cell：电芯编号
- microshortscore：ISC 综合评分 \[0,1\]
- microshortscore\_pct：评分百分位，代表所有内短路电芯中内短路的严重程度。比如某颗电芯microshortscore=0\.8，但是microshortscore\_pct=1，也就是它是最严重的那颗电芯。
- diagnosis\_result：诊断结果（microshortscore > 0\.75确诊，0\.6\-0\.75疑似，<0\.6正常）
- 诊断证据，已经归一化：s\_rank\_charge、s\_rank\_discharge、s\_hysteresis\_center、s\_charging\_ratio、s\_discharging\_ratio、s\_r0\_low

__示例问题与答案：__

1. 查询某 BMU 下某个电芯在指定时间窗口的 ISC 评分和诊断结果。

SELECT cell, microshortscore, diagnosis\_result, window\_id

FROM isc\_score\_result

WHERE bmu\_code = '目标BMU'

AND cell = 'CellVolt1'

AND window\_id = '20230201\_20230801';

1. 找出所有 ISC 综合评分大于 0\.8 的高风险电芯，按评分降序排列。

SELECT bmu\_code, cell, microshortscore, microshortscore\_pct

FROM isc\_score\_result

WHERE microshortscore > 0\.8

ORDER BY microshortscore DESC;

1. 统计每个 BMU 中微短路评分百分位高于 0\.9 的电芯数量。

SELECT bmu\_code, COUNT\(\*\) AS low\_pct\_count

FROM isc\_score\_result

WHERE microshortscore\_pct > 0\.9

GROUP BY bmu\_code;

### 3\. 电压温度异常结果表（volt\_temp\_abnormal\_result）

__表介绍__

- 记录模组在某一时刻的电压/温度异常事件。关键列：
- time：时间戳
- bmu\_code：模组编号
- type：异常类型（电压/温度）
- delta\_v：电压差值
- delta\_t：温差
- abnormal\_voltage\_cells / abnormal\_temp\_cells：异常单体信息（JSON 或字符串）
- volt\_value / temp\_value：异常电芯值列表

__示例问题与答案：__

1. 查询某个时间段内所有电压差值超过 0\.5V 的异常记录。

SELECT time, bmu\_code, delta\_v, abnormal\_voltage\_cells

FROM volt\_temp\_abnormal\_result

WHERE type = '电压'

  AND delta\_v > 0\.5

  AND time BETWEEN '2024\-01\-01' AND '2024\-01\-07';

1. 统计最近一个月温度异常（温差 > 10°C）最多的 BMU 编号。

SELECT bmu\_code, COUNT\(\*\) AS temp\_abnormal\_count

FROM volt\_temp\_abnormal\_result

WHERE type = '温度'

  AND delta\_t > 10

  AND time >= DATE\_SUB\(NOW\(\), INTERVAL 1 MONTH\)

GROUP BY bmu\_code

ORDER BY temp\_abnormal\_count DESC

LIMIT 1;

1. 列出所有异常电芯电压值大于 3\.6V 的记录，并显示对应时间。

SELECT time, bmu\_code, volt\_value

FROM volt\_temp\_abnormal\_result

WHERE JSON\_CONTAINS\(volt\_value, '3\.6', '$'\)   \-\- 假设 volt\_value 是 JSON 数组

   OR volt\_value LIKE '%3\.6%';                \-\- 若为字符串，用 LIKE

### 4\. 容量不一致电芯表（capacity\_inconsistent\_cells）

__表介绍__

记录被判定为容量不一致的电芯，包含自放电检测信息。关键列：

- bmu\_code：BMU 编号
- cell\_index：电芯索引
- first\_occurrence / last\_occurrence：首次/最近判定时间
- direction：异常方向（高/低/未知）
- has\_self\_discharge：是否存在自放电（0/1）
- maxvoltage\_drop\_rate\_mvh：最大电压下降率（mV/h）
- risk\_warning\_count：风险预警触发次数
- confidence\_score：置信度评分

__示例问题与答案：__

1. 查找某个 BMU 中异常方向为“低”且存在自放电的电芯。

SELECT cell\_index, direction, has\_self\_discharge, maxvoltage\_drop\_rate\_mvh

FROM capacity\_inconsistent\_cells

WHERE bmu\_code = '目标BMU'

  AND direction = '低'

  AND has\_self\_discharge = 1;

1. 输出所有最大电压下降率超过 5 mV/h 的电芯，并显示置信度评分。

SELECT bmu\_code, cell\_index, maxvoltage\_drop\_rate\_mvh, confidence\_score

FROM capacity\_inconsistent\_cells

WHERE maxvoltage\_drop\_rate\_mvh > 5

ORDER BY maxvoltage\_drop\_rate\_mvh DESC;

1. 统计每个 BMU 中容量风险预警次数大于 3 的电芯数量。

SELECT bmu\_code, COUNT\(\*\) AS high\_risk\_cell\_count

FROM capacity\_inconsistent\_cells

WHERE risk\_warning\_count > 3

GROUP BY bmu\_code;
